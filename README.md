A fully functional Pet Store Web Application built using Django.
The project allows users to browse pets/products, view details, add to cart, and explore an interactive UI similar to real e-commerce platforms.

* Key Features

  Beautiful Home Page with Banner & Carousel

  Search functionality

  Shopping Cart System

  User Login / Signup

  Product Listing with Images & Details

  Fully Responsive UI

  Clean & Attractive Design

  Tech Stack

Backend: Django (Python)

Frontend: HTML, CSS, Bootstrap

Database: SQLite / Django ORM

Tools: VS Code , Virtual Environment

* How to Run
git clone <repo-link>
cd petstore
python -m venv venv
source venv/Scripts/activate   (Windows)
pip install -r requirements.txt
python manage.py runserver


Open in browser:

http://127.0.0.1:8000/
